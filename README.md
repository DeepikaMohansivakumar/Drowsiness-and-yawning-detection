#  Driver Drowsiness Detection System

This project is a real-time driver drowsiness detection system using Python, OpenCV, and MediaPipe. It detects signs of fatigue like prolonged eye closure and yawning, and triggers an alarm to alert the driver.

---

##  Features

-  Eye Aspect Ratio (EAR) to detect closed eyes
-  Mouth Aspect Ratio (MAR) to detect yawning
-  Alarm sound when drowsiness is detected
-  Real-time webcam input
-  Simple and beginner-friendly implementation

---

##  Technologies Used

- Python 3
- OpenCV
- MediaPipe
- NumPy
- Pygame (for playing alert sounds)
